# Repete Auto Product

A polished Next.js landing-page prototype for Repete Auto in Vernal, Utah.

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Current scope

- One-page dealership landing page
- Brand-aligned yellow/black styling
- Contact, address, hours, inventory link, and vehicle finder pitch
- Static demo inventory cards
- Ready to expand into live inventory, lead capture, financing workflows, and admin tools

## Production notes

Do not hard-code vehicle prices as current unless the dealer approves them. Inventory changes quickly. The safer launch pattern is to link to the current inventory feed first, then integrate a real inventory source later.
